# NEET Study App

A comprehensive study companion designed for NEET aspirants, helping them organize their preparation, practice with mock tests, and access study materials.

## Features

* **Dashboard Overview**: View daily goals, quick access subjects, and preparation progress.
* **Study Materials**: Organized sections for Physics, Chemistry, and Biology, allowing structured learning.
* **Mock Tests**: Practice exams mimicking the NEET format to build exam readiness.
* **Progress Tracking**: Monitor preparation status and performance across subjects.
* **Responsive Design**: Works perfectly on mobile, tablet, and desktop devices.

## Tech Stack

* Flutter
* Dart

## How to Run

1. Clone the repository.
2. Ensure you have Flutter installed (`flutter doctor`).
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to launch the application.
